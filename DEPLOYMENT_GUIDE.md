## üîß Technical Details

### What the Workflow Does
1. **Checkout**: Gets your code from the `main` branch
2. **Setup Node**: Installs Node.js v20
3. **Install Dependencies**: Runs `npm ci` for clean install
4. **Generate**: Builds static site with `npm run generate`
5. **Deploy to gh-pages**: Pushes the `.output/public` directory to the `gh-pages` branch
6. **GitHub Pages**: Automatically serves the site from the `gh-pages` branch

### Branch Structure
- **`main` branch**: Your source code (Vue components, configs, etc.)
- **`gh-pages` branch**: Built static files (HTML, CSS, JS) - auto-generated by the workflow

### File Structure
```
.github/
  workflows/
    deploy.yml          # Automated deployment workflow

public/
  .nojekyll            # Prevents Jekyll processing

nuxt.config.ts         # Configured for GitHub Pages
```
